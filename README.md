# 📌 Project Overview

This project is a **PHP-based ATM Management System** that simulates the basic functionalities of a banking ATM.

Users can perform common banking operations such as:

- Account balance checking
- Cash withdrawal
- Fast cash
- Fund transfer
- PIN generation
- PIN change
- Mini statement
- Account summary

The project is built using **PHP, MySQL, HTML, and CSS** and runs on a **local server using XAMPP**.

---

# 🧠 System Features

| Feature | Description |
|------|------|
| PIN Login | Secure login using ATM PIN |
| Balance Inquiry | Check account balance |
| Cash Withdrawal | Withdraw money from account |
| Fast Cash | Quick withdrawal options |
| Fund Transfer | Transfer money between accounts |
| Generate PIN | Create new ATM PIN |
| Change PIN | Update existing ATM PIN |
| Mini Statement | View recent transactions |
| Account Summary | View account details |

# 💾 Database Setup

The project uses a **MySQL database**.

Database file:

```
atm.sql
```

Steps to import database:

1 Open **XAMPP Control Panel**

2 Start:

```
Apache
MySQL
```

3 Open browser and go to:

```
http://localhost/phpmyadmin
```

4 Click **New Database**

Create database:

```
atm
```

5 Click **Import**

6 Upload file:

```
atm.sql
```

Now database is ready.

# ⚙ Installation

## 🪟 Windows (Using XAMPP)

### 1 Install XAMPP

Download:

https://www.apachefriends.org

Install and open **XAMPP Control Panel**

Start:

```
Apache
MySQL
```

---

### 2 Copy Project Folder

Copy the **ATM project folder** into:

```
C:\xampp\htdocs\
```

Example:

```
C:\xampp\htdocs\atm-system
```

---

### 3 Import Database

Go to:

```
http://localhost/phpmyadmin
```

Create database:

```
atm
```

Import file:

```
atm.sql
```

---

### 4 Run the Project

Open browser:

```
http://localhost/atm-system
```
